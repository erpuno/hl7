SYNOPSIS

Minimalistic scalable microseconds precise FHIR application server in Elixir. ISO/HL7 27931:2009 is a simple HL7 application server implementation with the following features:
  • Fast (<5ms) JSON Schema Circular Validator RFC Draft-7;
  • MongoDB, RocksDB and Mnesia backends;
  • Erlang records as schema source for GRPC/JavaScript/Swift;
  • HTTP API using Plug;
  • FHIR Protocol Version R5;
  • Compact Codebase (10K LOC).
ISO/HL7 could be used as starter pack or SDK for medical systems.
